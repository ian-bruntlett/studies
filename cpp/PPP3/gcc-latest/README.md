# Information relevant to installing the gcc-latest C++ compiler

## Support files from my article

### irb-delete-elves
Bash shell script to delete ELF files - it asks permission before deleting a file. Typical usage:
./irb-delete-elves *

### prepare-for-gcc-latest
Shell script to set up environment for compiling C++ from the command line
